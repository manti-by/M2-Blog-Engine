Коратка пра сябе:

Вышэйшую тэхнічную адукацыю па спецыяльнасці інжынер-праграміст са спецыялізацыяй інфармацыйныя і інтэрнэт тэхналогіі.

Камунікабельнасць, уменне працаваць у камандзе, тэхнічны англійская, пісьменная гаворка, эмацыйная вытрымка, вялікі вопыт зносін з кліентамі (ад пенсіянераў да высокіх службовых асобаў), моцнае здароўе, без шкодных превычек.

Без праблем працуюць як на вокны платформы, так і на Linux платформы. Нядрэнна валодаю пакетам Adobe Photoshop.

Асноўныя звесткі:

  • Больш чым 12-гадовы практычны (8-гадовы камерцыйны) вопыт распрацоўкі вэб-прыкладанняў ад стадыі пастаноўкі задачы да запуску працуе праекта.
  • Высоконагруженных прыкладання са складанай архітэктурай і прапускной здольнасцю больш чым 5000rps. Гарызантальнае і вертыкальнае маштабаванне.
  • Веданне, разуменне і практычнае выкарыстанне ООП, шаблоны праектавання.
  • Пісьменнае афармленне кода (іерархія, апісанне, каментары), уменне чытаць і разумець чужой код, у тым ліку "Дрэнны", асінхронны і дэкампіляваць.
  • Аптымізацыя існуючага кода: Python, PHP, JS, SQL (Join / Union / Proc), HTML / CSS (кантэксты, шаблонизация, анімацыя), кэшаванне на розных узроўнях MVC. Інтэрфейсы і прототипирование, складанне тэхнічнай дакументацыі па праекце.
  • Адміністраванне і запуск сервераў: LAMP (Ubuntu, CentOS), WAMP (Windows Server). Apache / Nginx + Python, PHP (uWSGI, Gunicorn, Mod FCGI, CLI) + Memcache / Redis + MySQL / PostgreSQL / MongoDB, высокі ўзровень валодання Баш тэрміналам.
  • Залежнасці - Піп / Кампазітар / Новага сусьветнага парадку, адладка і прафіляванне кода, сістэмы кантролю версій - SVN / GIT / Mercurial, галінаванне, зліццё, адкаты і да т.п.
